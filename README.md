

### Первая работа с удаленным репозиторием

Работа с *commit*
Навыки на текущий момент,навык новых команд:
git remote add - связывает локальную и удаленную папки 
git pull - скачивает обновления
git push - отправляет код в сервер 
--no-rebase - для решения всех конфликтов сразу, а не для каждого сохранения отдельно
Навык склейки файлов, отправка текстового файла с жесткого диска в облако
